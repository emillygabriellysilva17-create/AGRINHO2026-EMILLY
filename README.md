
Este repositório reúne o projeto desenvolvido para o **Programa Agrinho 2026**, iniciativa educativa que conecta o meio rural e o meio urbano, destacando a importância da cooperação, da sustentabilidade e do desenvolvimento de soluções para o futuro.

## 🎯 Objetivos
- Mostrar a relação essencial entre campo e cidade;
- Apresentar como a produção rural abastece e sustenta os centros urbanos;
- Discutir práticas sustentáveis que beneficiam ambos os ambientes;
- Propor atividades educativas para alunos e comunidades.

## 📚 Conteúdos
- Contexto do Agrinho e sua importância;
- Fluxo de produtos, recursos e saberes entre campo e cidade;
- Desafios e soluções para um desenvolvimento equilibrado;
- Atividades práticas, pesquisas e trabalhos escolares.
