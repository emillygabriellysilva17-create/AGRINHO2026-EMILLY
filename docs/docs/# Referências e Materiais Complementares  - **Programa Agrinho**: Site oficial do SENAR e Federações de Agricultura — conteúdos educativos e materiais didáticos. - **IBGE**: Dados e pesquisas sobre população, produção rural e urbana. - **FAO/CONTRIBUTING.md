# Como Contribuir

Ficamos muito felizes com sua colaboração! Para participar:

1. Faça um **fork** deste repositório.
2. Crie uma nova branch com sua alteração: `git checkout -b minha-alteracao`.
3. Faça as alterações desejadas e adicione uma descrição clara do que foi feito.
4. Envie para o seu repositório: `git push origin minha-alteracao`.
5. Abra um **Pull Request** explicando sua contribuição.

Sugestões, correções, novos conteúdos e materiais educativos são sempre bem-vindos!

Por favor, mantenha o conteúdo alinhado ao tema do Agrinho 2026 e respeite as diretrizes educacionais e éticas.
